Instruções:

Esta versão do jogo pacman consiste numa alteração do jogo original, introduzindo algumas habilidades especiais.

Objetivo

O objetivo deste jogo é comer todas as bolas brancas, sem colidir com os fantasmas. Esta tarefa é facilitada com o uso de bolas especiais que atribuem certas habilidades ao pacman.

Habilidades:

Invert: É ativada quando o pacman passa nas bolas vermelhas. Inverte a trajetória do fantasma vermelho e azul, e torna o cenário vermelho.
SpeedUp: É ativada quando o pacman come a bola verde. O pacman ganha super velocidade, tal como os fantasmas.
Godmode: É ativada quando o pacman come a bola roxa. O pacman torna-se invulnerável temporariamente, podendo atravessar fantasmas enquanto estiver sob o efeito da bola roxa.

Controlos:

Seta para cima: Pacman move-se para cima;
Seta para baixo: Pacman move-se para baixo;
Seta para a direita: Pacman move-se para a direita;
Seta para a esquerda: Pacman move-se para a esquerda;
Tecla 'm': Ligar/Desligar o som;
Tecla 'Spacebar': Colocar o jogo em pausa;
